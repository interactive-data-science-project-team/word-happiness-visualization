Completeness

Are there junk values or missing values? There are some missing entries (na) in the original dataset (see table below), especially for GINI index (63% of the data do not have the corresponding World bank estimated GINI index). Some survey questions are not asked in certain countries. For example, the perception of corruption is not asked in China. This results in missing values for these attributes for all years in some countries. This may introduce some bias into the data when we perform further analysis using techniques like regression analysis. Other columns have few missing entries.

We examined the number of reports each year by country, and we find that there are significantly less reports collected before 2007 (see the table below).

How did we handle missing entries? We did preliminary data cleaning using pandas. If we have missing data for an attribute, we try to use the last valid observation from the same country to fill in the missing values, df.groupby('country', as_index=False).fillna(method='ffill'). If this fails, we try to use the next valid observation to fill in the missing values, df.groupby('country', as_index=False).fillna(method='bfill'). It's possible that some attributes were not collected for some countries for the entire time period such as the "perception of corruption" attribute for China. In such cases, we leave the values as na. The table below show the number of na 's before and after the data cleaning.
